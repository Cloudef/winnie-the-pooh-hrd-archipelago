# Winnie the Pooh's Home Run Derby

## Things to know:

- Bonus points cannot be earned, all Pooh's power ups are items
- Stages itself can also be unlocked as items (enabled with `shuffle_stages` setting)
- If using the above option, you may want to specify `start_inventory: {Eeyore: 1}` so the game isn't BK'd from the start

## What is randomized?:

- Pooh's power ups
- Stages (`shuffle_stages` setting)

## What are the locations?

- Stages have multiple checks (amount depends on settings, but usually around 8-10)
- If you do a perfect clear, you unlock all checks for the stage
- If you get 2x the target amount, you unlock 3 checks for the stage at once

## What is the goal?

Defeat Christopher Robin
