# Winnie the Pooh's Home Run Derby Setup Guide

## Required Software

* A copy of [Winnie the Pooh's Home Run Derby](https://en.wikipedia.org/wiki/Winnie_the_Pooh%27s_Home_Run_Derby) (English swf)
* An installation of [Archipelago](https://archipelago.gg/tutorial/Archipelago/setup_en)
* [Ruffle Flash Player](https://ruffle.rs/)

## Setup

1. Create a seed
2. Host a server
3. Start the game from the Archipelago
4. Select the swf file (client remembers the path)
5. Select the ruffle executable if on windows (client rememebers the path) (on non-windows ruffle is assumed to reside within PATH)
6. Enjoy!
